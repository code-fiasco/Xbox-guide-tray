Initial release of Xbox Guide Tray.

## Highlights

- Guide button short press toggles your configured game launcher (Playnite, Steam Big Picture, etc.)
- Guide button long press opens a controller-friendly power menu
- Bluetooth controller pairing, monitoring, and disconnect from the tray
- Optional HidHide integration to block controller input from reaching other apps while the power menu is open
- Windows installer with optional HidHide bundling and uninstall prompt

## Requirements

- Windows 10 or later (64-bit)
- Bluetooth-capable PC
- Xbox controller paired over Bluetooth

## Install

Download `XboxGuideTray-Setup-1.0.0.exe` below and run the installer. On first launch, open Settings to choose your controller and launcher application.
